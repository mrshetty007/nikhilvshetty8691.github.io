INVENTORY DASHBOARD

Project Overview

This dashboard was created to monitor the status of assets in the inventory across different countries, as well as identify state and substate errors. It incorporates numerous processes, such as gathering depot data from all locations and comparing it with our database to determine the correct state and substate of the assets. This comprehensive approach ensures that the data is up-to-date and accurate, providing real-time insights into the inventory status. Additionally, the dashboard features intuitive visualizations and user-friendly interfaces, making it easier for team members to access and understand the data. This tool has significantly improved data accuracy from 65% to 95% and reduced manual effort by allowing the team to efficiently extract and analyze data directly from the dashboard. Consequently, it has streamlined operations and enhanced decision-making processes within the organization.

Key Responsibilities

. To Identify the asset current state and substate .

. To keep all the asset attributes in co-ordination with the depot and Asset management team .

. To find the error percentage and error of the asset and correct the data to maintain data accuracy .

. To Avoid manual work in preparartion of data and extract it from dashboard.

Process
**Error Identification**:

. The dashboard features an error tab view that uses bookmarks and selection to display different types of errors. This allows users to quickly identify and address issues.

. Errors are categorized by types such as "Recently received," "Lastshipdate is Late," and "Substatus Mismatch," among others.

. An example of this can be seen in the error split visualization, which shows the count of each error type, enabling targeted error resolution (as shown in the first image).

**DAX Equations**:

. The dashboard utilizes DAX (Data Analysis Expressions) equations to implement complex logic for data validation.

. These DAX equations are used to compare the depot data with the database and determine the correct state and substate of the assets.

. The equations handle various conditions and scenarios to ensure data accuracy, such as checking for pending disposals, repairs, evaluations, and more (as demonstrated in the second image).

**Data Visualization**:

. The main dashboard view provides a comprehensive visualization of asset status, including substate breakdown, GL account breakdown, model count, and error indicators.

. It features various charts and metrics to offer a detailed and interactive view of the inventory data.

. Key metrics include the overall error count, asset error count, and accuracy percentages, which are crucial for monitoring the performance and reliability of the asset data (as shown in the third image).

Result

The Asset Inventory Dashboard significantly improved the accuracy and reliability of the asset data. Automation through Power BI not only reduced errors but also saved considerable time and resources for the team. By leveraging advanced DAX equations and dynamic visualizations, the dashboard provided a powerful tool for asset management and data validation, contributing to enhanced operational efficiency.
