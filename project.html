DNA VALIDATION USING PYTHON

Project Overview

The DNA Validation Process project involved developing a comprehensive system/code using data manipulation python libraries such as numpy pandas to accurately validate and analyze assets DNA sequences. The primary objective was to ensure the integrity and correctness of DNA data of the assets, which is crucial for asset activity and its lifecycle. The output of this code gives multiple files by comparing discovery files and assets file. This code is developed using Prompt Engineering.

Key Responsibilities

• Implemented algorithms to detect and correct errors in DNA sequences, improving data reliability.

• Using AI tools (ChatGPT) and prompt engineering to improve the efficiency of the code.

• Using numpy and pandas data libraries to clean and manipulate the data.

• Using proper DNA Logic by studying the DNA process from the Client's handbook and implementing the logic into the code.

• Checking the process or the output files to check the data accuracy.
